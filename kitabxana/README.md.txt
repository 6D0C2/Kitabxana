# 📚 Kitabxana İdarəetmə Sistemi

FastAPI ilə yazılmış sadə kitab idarəetmə tətbiqi.

## Əməliyyatlar
- Kitabların siyahısını görmək
- Yeni kitab əlavə etmək
- Kitab məlumatlarını yeniləmək
- Kitab silmək
- ID ilə kitab axtarmaq

## Quraşdırma

### Tələblər
- Python 3.9+

### İşə Salma
pip install -r requirements.txt
py -m uvicorn main:app --reload

Brauzderdə açın: http://localhost:8000
